# coldpdf

a simple browser-based pdf tool. everything runs locally in your browser using web workers, so no files are uploaded to any server.

## tools included
- merge pdfs
- split & delete pages
- reorder & rotate
- draw/mark on pdf
- convert to/from images (jpg, png)
- compress
- password protect / unlock
- watermark & redact
- ocr / text extraction
- dark mode & invert colors

## running locally
no build step required. just open `index.html` in your browser.
or use python:
`python -m http.server 8000`

## tech stack
- pdf-lib
- pdf.js
- vanilla js & css
- sortablejs
